# Gitea Pull Request and Review Guide

Follow these steps to submit a pull request as Max and approve it as Tom.

### 1. Create the Pull Request (As Max)
* Log into the Gitea web interface using **Max's** credentials.
* Navigate to your project repository.
* Click on the **Pull Requests** tab and select **New Pull Request**.
* Choose your working branch and compare it against the main branch.
* Click **Create Pull Request**, fill out the title, and assign **Tom** as the reviewer in the right-hand panel.

### 2. Approve the Pull Request (As Tom)
* Log out of Max's account and log back in using **Tom's** credentials.
* Navigate to the **Pull Requests** tab of the repository to find Max's request.
* Click on the **Files Changed** tab to inspect the modifications.
* Click the **Review Changes** button, select **Approve**, and click **Submit review**.
